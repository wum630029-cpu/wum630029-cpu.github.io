## 一、什么是空投？为什么项目要发空投？

### 一句话理解

空投就是项目方**免费送代币**给符合条件的钱包地址。条件可以是在某个时间点持有某种代币、完成了特定的链上交互、或者参与了社区活动。

### 项目方的动机

| 动机 | 说明 |
|:----|:-----|
| **激励早期用户** | 奖励那些在项目早期就参与测试、提供流动性的真实用户 |
| **去中心化治理** | 把治理代币分发到大量用户手中，避免权力过度集中 |
| **市场推广** | 空投本身就是最大的营销事件——免费代币的消息能迅速引爆社区 |
| **流动性引导** | 用户收到代币后，部分人会选择持有或提供流动性，间接为项目赋能 |

## 二、空投的五大类型

### 1. 快照空投（Snapshot Airdrop）

项目方在某个时间点对链上状态进行快照，根据地址持有的代币或 NFT 数量分配空投。

**特点：** 无需操作，持有即赚。适合持有核心资产（ETH、稳定币）或项目方发行的 NFT。

**典型案例：** Uniswap（向所有历史用户快照）、以太坊分叉空投。

### 2. 交互空投（Interaction Airdrop）

根据用户在项目协议上的操作记录分配空投——交易次数、金额、时间跨度等都是评判标准。

**特点：** 需要主动操作，是当前最常见的空投类型。**也是本指南的重点。**

**典型案例：** Arbitrum（要求主网交互）、zkSync（要求 Era 主网交互）、Scroll、Linea。

### 3. 测试网空投（Testnet Airdrop）

在项目上主网之前，鼓励用户在测试网上参与体验。测试网代币免费，交互成本为零。

**特点：** 零成本，但需要把握时机——在主网上线前完成交互。适合早期发现的新项目。

**典型案例：** LayerZero、zkSync 早期测试网参与者。

### 4. 任务空投（Quest/Points Airdrop）

通过 Galxe、Zealy、Layer3 等任务平台完成指定任务（关注推特、加入 Discord、链上交互等），积累积分，最终按积分兑换代币。

**特点：** 操作简单，有明确的任务清单，但耗时长、单个任务权重低。

**典型案例：** Linea Voyage、zkSync Era 活动、Scroll Marks。

### 5. 质押/存款空投（Staking/Deposit Airdrop）

将资产存入项目协议（通常为流动性质押或借贷），根据存款金额和时间获得空投权重。

**特点：** 资金占用成本高，但权重更大。适合有闲置资金且熟悉 DeFi 操作的用户。

**典型案例：** EigenLayer（再质押空投）、Blast（跨链存款积分）。

### 3.2 Gas 费管理

交互空投最大的成本就是 Gas 费。2026 年以太坊主网 Gas 费通常在 5-50 gwei 之间波动。

**Gas 优化技巧：**
- **避开高峰期**：亚洲工作日晚 8-11 点 Gas 最贵，周末早晨相对便宜
- **使用 Layer 2**：在 Arbitrum、Base、Optimism 上交互，Gas 费只有主网的 1/10 甚至更低
- **批量操作**：一次调用多个合约，减少重复交易
- **Gas 追踪工具**：使用 [Etherscan Gas Tracker](https://etherscan.io/gastracker) 或钱包内置的 Gas 建议功能

### 3.3 资金规划

- **推荐启动资金**：0.1-0.5 ETH（或等值 USDC/USDT）
- **分配方式**：不要把所有资金放在一个地址。每个地址 0.02-0.1 ETH 就够了
- **稳定币**：准备一些 USDC/USDT 用于在 DEX 上做交易交互

### 3.4 跨链桥准备

在多个链之间交互时，需要用到跨链桥。参考本站的 [《跨链桥与多链生态操作指南》](/cross-chain-bridge-guide/)。

| Relay.link | 跨链速度快，体验好 |
| Across | 以太坊主网 ↔ L2 高效 |

## 四、2026 年热门空投交互策略

空投交互的核心逻辑是：**模拟真实用户的高质量链上行为。** 项目方（和女巫检测算法）看的是交互的质量和自然度，而不是单纯的交易量。

### 4.1 基础交互模型

无论哪个项目，核心交互操作都包括以下几类：

```
✅ 必须做的操作：
   1. 多次 SWAP（兑换）——不同代币对、不同金额
   2. 添加/移除流动性
   3. 借贷（存款 + 借款）
   4. 跨链操作（如果有跨链场景）
   
✅ 加分操作：
   5. 持有项目方的 NFT 或参与社区治理
   6. 在不同时间段内多次交互（至少拉长到 2-4 周）
   7. 使用项目方的前端，而非聚合器
   
❌ 减分/危险操作：
   8. 所有地址在同一时间做完全相同的操作序列
   9. 单笔交易后立即撤出所有资金
   10. 使用同一个 CEX 提现地址给所有交互地址打钱
```

### 4.3 以 Scroll 为例的交互实操

Step-by-step 教学：

**Step 1：跨链到 Scroll**

1. 每周操作 2-3 次，持续 1-3 个月
2. 不要在同一天内完成所有操作
3. 每次金额不要完全相同

## 五、女巫检测与防女巫策略

这是空投交互中最关键也最容易忽视的部分。所谓「女巫攻击」（Sybil Attack），是指同一人创建大量虚假账户来骗取空投份额。项目方对此深恶痛绝，会使用越来越复杂的算法来识别女巫地址。

### 女巫检测的常见手段

| 检测维度 | 检测方法 |
|:--------|:--------|
| **资金来源** | 多个地址从同一个 CEX 地址提现 → 极大可能被标记为女巫 |
| **交互模式** | 所有地址以完全相同的顺序调用相同的合约 → 机器人行为 |
| **时间模式** | 所有交互集中在短时间内完成 → 非自然行为 |
| **金额模式** | 每个地址的交互金额几乎相同 → 非自然行为 |
| **合约交互** | 只和项目方合约交互，不和任何生态应用交互 → 狩猎行为 |
| **网络关联** | 多个地址使用同一个 IP 或浏览器指纹 → 关联地址簇 |

### 如何降低被标记的风险

```
✅ 推荐做法：
   - 每个地址使用独立的新钱包（不要导入到同一个钱包扩展中）
   - 每个地址从不同的 CEX 地址或不同的入口打钱
   - 交互金额有所差异（0.02、0.05、0.1 ETH 混着用）
   - 拉长时间跨度，分散操作
   - 除了核心协议外，也交互一些生态小应用
   - 持有一些生态 NFT（增加真实用户权重）

❌ 千万不要做：
   - 10 个地址在同一小时做完全相同的 5 笔交易
   - 所有地址从同一个币安提现地址接收初始资金
   - 批量创建钱包、批量操作 —— 这不是交互，这是「狩猎」
```

## 七、空投防骗指南

空投领域骗子极多。2026 年的常见空投骗局包括：

### 🔴 骗局类型

| 骗局 | 手法 | 如何识别 |
|:----|:----|:--------|
| **假空投网站** | 模仿知名项目的网站，让你连接钱包并签名 | 核对官方域名！检查 SSL 证书、社交媒体链接 |
| **钓鱼签名** | 诱导你在「领取空投」时代理钱包权限 | 仔细阅读每一笔签名内容，看不懂的不签 |

| **Discord 冒充** | 冒充项目管理员私信你「领空投」 | 真正的管理员永远不会私下联系你 |
| **虚假账号** | 在 Twitter 上冒充项目官方发布假链接 | 检查账号认证状态、创建时间、粉丝互动 |

## 八、常见问题 FAQ

### Q1：空投需要多少钱启动？

最少 0.1 ETH（约 2000 元人民币）就够在 1-2 个地址上做基础交互。更理想的启动资金是 0.5-1 ETH，可以在 5-10 个地址上运作。注意这些都是 Gas 费和流动性成本——不是打水漂，资金最终可以收回。

### Q2：撸空投需要懂编程吗？

不需要。2026 年的空投交互工具已经非常成熟，基本上只需要会使用 MetaMask、DEX、跨链桥等基础工具即可。会编程是加分项（可以批量分析链上数据、自动交互），但不是必要条件。

### Q3：一个项目需要交互多久？

建议至少交互 1-3 个月。从项目主网上线开始交互，持续到快照或官方宣布空投规则为止。数据显示，交互持续 1 个月以上的地址获得空投的概率远高于只交互几天的地址。

### Q4：用什么链交互成本最低？

Layer 2 上交互最为划算。Arbitrum、Base、Optimism 上的 Gas 费通常只有以太坊主网的 1/10-1/20。Scroll、Linea 也相对便宜。但有些空投项目（如 L1 新公链）仍需要在主网上操作。

### Q5：空投收益需要交税吗？

在大多数国家（包括美国、欧盟、日本），空投收入被视为应税收入，收到时的公允价值需要申报。加密货币税收较为复杂，建议咨询专业税务师或参考本站 [《加密货币税务指南》](/crypto-tax-guide/)。

### Q6：2026 年撸空投还值得吗？

相比 2020-2022 年的「黄金期」，2026 年的空投收益已经下降，但仍然是一个不错的低门槛切入点。关键在于策略：不再盲目「广撒网」，而是精选高潜力项目做深度交互。一个成功的空投（如 EIGEN、ZK）仍然可以带来数千甚至上万美元的收益。
