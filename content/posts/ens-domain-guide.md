> 如果你的钱包地址是 `0xAbc...123`，有没有想过把它变成 `yourname.eth`？——每次转账不用复制粘贴长长的一串十六进制字符，别人看到你的时候也知道你是谁。
>
> **这就是 ENS（Ethereum Name Service）做的：把人类可读的名字和复杂的区块链地址绑定起来。** 它不仅仅是「地址缩短器」，更是你在 Web3 世界的**身份名片**。

## 一、什么是 ENS？

### 一句话理解

**ENS 是「以太坊的域名系统」——就像互联网的 DNS 把 `google.com` 指向服务器 IP 地址一样，ENS 把 `yourname.eth` 指向你的钱包地址。**

但 ENS 能做的远不止地址解析：

| 功能 | 说明 | 类比 |
|:----|:----|:----|
| **地址解析** | `yourname.eth` → `0xAbc...` | 像 DNS 把域名解析成 IP |
| **反向解析** | `0xAbc...` → `yourname.eth` | 别人看到你的地址时显示你的名字 |
| **多币种地址** | 一个 ENS 名绑定 BTC、ETH、SOL 等多个链的地址 | 不用记不同链的地址 |
| **子域名** | `pay.yourname.eth`、`nft.yourname.eth` | 像子域名管理 |
| **文本记录** | 绑定 Twitter、邮箱、头像、网站等 | 链上名片 |
| **内容哈希** | 指向 IPFS 网站或文件 | 去中心化网站的基础 |

### 为什么你需要 ENS？

- **好记**：`yourname.eth` 比 `0x7A3...9F2` 容易记一百倍，转账不易出错
- **专业**：在 DApp 交互、空投领取、社区讨论中展示统一的链上身份
- **多功能**：一个 ENS 可以绑定多个链的地址，跨链收款更方便
- **可移植**：ENS 是 NFT 形式存在的资产，可以转让、出售、作为数字收藏品
- **投资属性**：优质短域名有升值空间（类似早期的域名投资）

### ENS 与传统 DNS 的区别

| 对比项 | ENS | DNS |
|:------|:---:|:---:|
| 运行平台 | 以太坊区块链 | 中心化服务器 |
| 所有权 | 你拥有私钥 = 拥有域名 | 你租用域名，需续费 |
| 是否可被审查 | ❌ 不可（除非以太坊被关闭） | ✅ 可被注册局/政府没收 |
| 到期机制 | 注册后永久持有，但需续租 | 每年续费，不续费被收回 |
| 顶级域名 | `.eth`（ENS 原生）+ 支持 DNS 域名导入 | .com、.org 等数百种 |
| 去中心化程度 | 完全去中心化 | 依赖于 ICANN 等中心化机构 |

## 二、ENS 域名命名规则与价格

### 命名规则

| 规则 | 说明 |
|:----|:----|
| 长度 | 3 字符及以上（3 位最短，价格最贵） |
| 可用字符 | `a-z`、`0-9`、以及连字符 `-`（不可在开头或结尾） |
| 特殊字符 | 不支持大写字母、空格、符号 |
| 热门词 | 常见英文单词、数字组合已被大量注册 |

> 💡 中文 .eth 域名现在也支持了——例如 `你的名字.eth`，通过 Punycode 编码实现。但实际使用中，英文/数字域名兼容性更好。

### 价格与续费

ENS 采用**按年续租**模式（不是一次买断）：

| 字符长度 | 注册费（年） | 续费（年） | 说明 |
|:-------|:----------:|:---------:|:----|
| 3 字符 | ≈ 640 ETH/年 | ≈ 640 ETH/年 | 极其昂贵，通常是拍卖获得 |
| 4 字符 | ≈ 160 ETH/年 | ≈ 160 ETH/年 | 较贵，稀有品 |
| 5 字符 | ≈ 1.5 ETH/年 | ≈ 1.5 ETH/年 | 中等价格 |
| 6+ 字符 | **≈ $5–15 USD/年** | **≈ $5–15 USD/年** | 普通用户的首选 |

> ⚠️ 以上价格以 ETH 计价，会随 ETH 价格波动。实际支付时使用 ETH 或 USDC。2026 年，一个 6 字符以上的 .eth 域名年费大约在 $5–15 之间。

**重要提示**：
- ENS 不是一次买断——每年都需要续费
- 超过 90 天宽限期不续费，域名会被释放
- 释放后任何人都可以注册
- 可以使用**多年代付**（最多 10 年）锁定当前价格

## 三、注册 .eth 域名的完整流程

### 3.1 准备工作

在开始注册前，你需要：

1. ✅ **一个 Web3 钱包**（推荐 [MetaMask](/metamask-guide/)）
2. ✅ **钱包中有 ETH**（用于支付注册费和 Gas 费—主网交互需要 ETH）
3. ✅ **一个可用的 .eth 域名想法**（想好你想要的名字）
4. ✅ **建议在 Gas 较低时段操作**（北京时间凌晨 2–8 点或周末）

### 3.2 检查域名是否可用

1. 打开 [app.ens.domains](https://app.ens.domains)——ENS 的官方管理页面
2. 在搜索框中输入你想要的名字（不需要带 `.eth`）
3. 系统会告诉你：
   - ✅ **Available**：可以注册
   - ❌ **Already registered**：已被注册
   - 🔒 **For auction**：3–4 字符域名需通过拍卖获取

> 💡 **搜索技巧**：一个 6–8 位的短域名虽然可能被注册，但很多过期域名会陆续释放。你可以关注 [ENS 过期域名列表](https://dune.com/makoto/ens) 来捡漏。

### 3.3 注册步骤详解

#### 第一步：发起注册请求

1. 输入你的理想域名，确认 Available
2. 点击 **Register** 按钮
3. 连接你的钱包（MetaMask 会自动弹出连接请求）
4. 选择注册年限（1–10 年—推荐至少注册 2 年，避免频繁续费）

#### 第二步：提交注册交易（共两笔）

ENS 注册需要发送**两笔链上交易**：

**交易 1：请求注册（Commit——约 30–60 秒）**
- MetaMask 会弹出一个签名请求
- 这笔交易是「锁定你的注册意愿」，防止别人提前发现你想注册什么域名
- Gas 消耗较小（约 $2–5）

> 🔧 **为什么需要两笔交易？**
> ENS 采用「Commit-Reveal」机制防止域名抢注。简单来说：
> - Commit 阶段：你提交一个哈希值，隐藏了你真正想注册的域名
> - Reveal 阶段：解锁并完成注册
> - 这样其他人无法通过观察内存池来抢注你正在注册的域名

**交易 2：完成注册（Register——等待 60 秒后）**

等待约 60 秒（1 个以太坊区块时间）后：

1. 页面上的「Register」按钮会变为可点击
2. 点击完成注册
3. MetaMask 弹出第二笔交易
4. 确认并支付 Gas（这笔 Gas 费用更高，约 $10–30，取决于网络拥堵情况）
5. 等待交易确认（约 1–5 分钟）

#### 第三步：注册成功

- 🎉 交易确认后，你就拥有了 `yourname.eth` 的所有权
- 该域名会以 NFT（ERC-721）的形式出现在你的钱包中
- 你可以在 OpenSea 等 NFT 市场上看到它
- 现在你就可以设置解析记录了

### 3.4 设置解析记录

注册完成后，最重要的一步：**将你的 ENS 域名指向你的钱包地址。**

1. 在 [app.ens.domains](https://app.ens.domains) 搜索你的域名并点击进入
2. 点击域名详情页的 **Records**（记录）标签
3. 你会看到以下可设置的记录类型：

#### 核心设置：ETH 地址解析

1. 在「Ethereum Address」字段输入你的钱包地址
2. 点击 **Save** 按钮
3. 钱包弹出签名请求，确认发送交易（需要支付 Gas）
4. 交易确认后，别人就可以通过 `yourname.eth` 向你转账了

#### 进阶设置：多币种地址

点击 **Add Record** → **Address**，你可以绑定其他链的地址：

| 链 | 格式 | 示例 |
|:--|:----|:----|
| BTC | 比特币地址 | `1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa` |
| SOL | Solana 地址 | `7EcDhSYGxXyscszYEp35KHN8vvw3svAuLKTjW8qSgUCV` |
| MATIC | Polygon 地址 | `0x...`（和以太坊地址格式相同） |
| BNB | BNB Chain 地址 | `0x...`（同上） |
| OP | Optimism 地址 | `0x...`（同上） |

> 💡 **设置多链地址后，别人可以一次查询你的所有收款地址**——无论是转账 ETH 还是 BTC，都只需要知道你的 `yourname.eth`。

#### 社交记录

点击 **Add Record** → **Text**，可以绑定：

| 记录类型 | 字段 | 用途 |
|:--------|:----|:----|
| 头像 | `avatar` | 你的 ENS 头像（在支持 ENS 的 DApp 中显示） |
| 网址 | `url` | 你的网站或个人主页 |
| 邮箱 | `email` | 联系邮箱 |
| Twitter | `com.twitter` | Twitter ID |
| GitHub | `com.github` | GitHub 用户名 |
| 钱包备注 | `note` | 自定义备注信息 |

### 3.5 设置反向解析

反向解析指的是：**当别人查询你的钱包地址时，能看到你的 ENS 名称。**

1. 在域名详情页点击 **Reverse Record**（反向记录）
2. 点击 **Add Reverse Record**
3. 选择你的 ENS 域名作为反向解析
4. 钱包确认交易

完成这一步后，当你在支持 ENS 的 DApp 中被查看地址时，会显示 `yourname.eth` 而不是 `0x...`。

> 💡 很多 DeFi 协议（Uniswap、1inch）、钱包（MetaMask、Rainbow）、甚至区块浏览器（Etherscan）都支持 ENS 反向解析——设置后你的身份在这些平台上都会统一显示。

## 四、ENS 子域名管理

**子域名（Subdomain）** 是 ENS 最强大的功能之一。

### 什么是子域名？

类似 `mail.google.com` 是 `google.com` 的子域名——`pay.yourname.eth` 是 `yourname.eth` 的子域名。

### 子域名的用途

| 用途 | 示例 | 说明 |
|:----|:----|:----|
| 家庭成员 | `alice.yourname.eth` | 给家人的钱包分配一个子域名 |
| 按用途分类 | `pay.yourname.eth`、`nft.yourname.eth` | 不同用途使用不同钱包地址 |
| 社区成员 | `member.dao.eth` | 给社区成员分配身份 |
| 企业架构 | `dev.company.eth`、`finance.company.eth` | 企业内部部门标识 |
| 临时地址 | `event2026.yourname.eth` | 临时活动的收款地址 |

### 如何创建子域名

1. 打开 [app.ens.domains](https://app.ens.domains) → 进入你的域名
2. 点击 **Subdomains**（子域名）标签
3. 输入子域名名称（例如 `pay`）
4. 设置子域名的解析记录（指向哪个钱包地址）
5. 确认交易并支付 Gas

**子域名的特点**：
- 子域名的**所有者**可以独立管理自己的解析记录
- 主域名所有者可以随时收回子域名
- 子域名**不需要单独续费**——只要主域名有效，子域名就有效
- 每个子域名的创建都需要 Gas 费

> 💡 **实用场景**：如果你有一个 DAO 或社区，可以为每个成员创建一个子域名（如 `member.address.eth`）作为其链上身份。这个子域名可以绑定到成员的私人钱包地址，即使成员更换钱包，只需要你更新子域名的解析记录即可，对外仍显示同一个名字。

## 五、ENS 进阶用法

### 5.1 在钱包中使用 ENS

大多数主流钱包已经原生支持 ENS：

| 钱包 | ENS 支持程度 |
|:----|:-----------|
| **MetaMask** | 输入 `.eth` 地址自动解析，显示反向解析名称 |
| **Rainbow** | 原生 ENS 支持，界面友好 |
| **ENS Domains** | 专为 ENS 打造的钱包，自动使用你的 .eth 作为身份 |
| **Zerion** | 支持 ENS 解析和展示 |
| **Rabby** | 支持 ENS 地址输入 |
| **OKX Web3 钱包** | 支持 ENS 解析 |

**实际操作**：在 MetaMask 的发送页面输入 `vitalik.eth`——会自动解析为 `0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045`。

### 5.2 ENS 与去中心化网站

ENS 可以指向 IPFS 或 IPNS 内容哈希，从而搭建去中心化网站：

1. 设置内容的 `contenthash` 记录为你的 IPFS CID
2. 通过 `yourname.eth.link` 或 ENS 网关访问
3. 你的网站就运行在 IPFS 上——没有中心化服务器，无法被封禁

**适合人群**：
- 不希望网站被审查的内容创作者
- 去中心化应用的着陆页
- 个人去中心化博客

### 5.3 ENS 注册局（DNS 导入 ENS）

如果你已经在 DNS 注册了一个传统域名（如 `yourdomain.com`），你可以：

1. 在 ENS App 中选择 **DNS Import**
2. 在 DNS 设置中添加指定的 TXT 记录来验证所有权
3. 验证通过后，`yourdomain.com` 就可以像 `.eth` 一样使用 ENS 功能

> ✅ **优势**：不用额外注册 `.eth` 域名，直接用你的 .com 域名享受 ENS 功能。
> ⚠️ **限制**：需要至少支持 DNSSEC 的 DNS 服务商，且过程比较复杂。

### 5.4 ENS 交易市场

ENS 域名本身是 NFT，可以在 NFT 市场上交易：

| 平台 | 特点 |
|:----|:----|
| **OpenSea** | 最大的 ENS 二级市场，可直接搜索 .eth 域名 |
| **ENS.Market** | 专门的 ENS 交易市场，支持批量操作和过期域名监控 |
| **NameStone** | 社区拍卖平台，用于高价域名交易 |

**哪些 .eth 域名价值高？**
- 3–4 字符短域名（最稀缺）
- 常见英文单词、姓氏（如 `wang.eth`、`eth.eth`）
- 数字域名（如 `888.eth`）
- 品牌相关域名
- 有流量的域名（有反向链接和关注者）

## 六、Gas 优化技巧

注册和管理 ENS 需要在以太坊主网上支付 Gas 费，以下技巧可以帮你省钱：

### 6.1 选择低 Gas 时段

以太坊的 Gas 价格随时间波动：

| 时段 | Gas 价格 | 建议 |
|:----|:-------:|:----|
| 北京时间凌晨 2–8 点 | 🟢 低 | ✅ 最佳注册时间 |
| 北京时间白天（工作日） | 🟡 中 | 可接受 |
| 热门项目发币/大事件时 | 🔴 高 | ❌ 避免操作 |
| 周末（特别是周日） | 🟢 较低 | ✅ 推荐 |

**Gas 查询工具**：
- [etherscan.io/gastracker](https://etherscan.io/gastracker)——以太坊官方 Gas 追踪
- [gasnow.org](https://gasnow.org)——各链实时 Gas 数据
- [app.ens.domains](https://app.ens.domains) 注册页面也会显示当前 Gas 估算

### 6.2 使用批量操作

如果你需要注册多个域名或设置多个记录，建议：

- **一次性完成所有设置**：一次链上交易可以包含多个操作，比多次单笔交易便宜很多
- **使用 ENS App 的 Bulk Operations**：ENS 官方支持批量注册域名
- **子域名批量创建**：设置好模板后，批量创建子域名节省平均 Gas

### 6.3 提前续费多年

ENS 支持一次续费最多 10 年：

- 一次性续费多年（如 5 年），只需要**一次链上交易**的 Gas
- 相比每年续费一次 × 5 次，总共节省 4 次 Gas 费
- 还能锁定当前的续费价格——如果 ETH 涨价，你实际上赚了

### 6.4 使用 Layer 2 部署吗？

目前 ENS 的**注册和管理**必须在以太坊主网上进行（L1）。但有趣的是，**ENS 域名的使用**在 L2 上也完全受支持：

- 在 Arbitrum、Optimism、Base 上转账时输入 `.eth` 地址同样可以解析
- 但注册新域名和设置解析记录需要回到主网

> 💡 好消息是 ENS 团队正在推进多链部署（ENSv2），未来可能在 L2 上直接注册和管理域名。

## 七、安全注意事项

### 7.1 续费提醒——不要丢失你的域名

**这是 ENS 持有者最常见的失误：忘记续费。**

| 时间线 | 状态 | 后果 |
|:------|:----|:-----|
| 到期日前 | ✅ 正常使用 | 无影响 |
| 到期后 0–90 天 | ⚠️ **宽限期** | 域名仍然可用，续费即可 |
| 到期后 90–120+ 天 | 🚫 **释放期** | 域名不可用，但仍可续费（费用更高） |
| 被释放后 | ❌ **域名丢失** | 谁都可以重新注册 |

**防丢失建议**：
1. 设置日历提醒，提前 30 天提醒续费
2. 一次性续费 5–10 年（省 Gas 也省心）
3. 使用 ENS App 的通知功能（注册时留下邮箱）
4. 关注 ENS 官方的续费提醒邮件

### 7.2 常见钓鱼手法

| 钓鱼手法 | 特征 | 防范措施 |
|:--------|:----|:--------|
| **假注册网站** | 模仿 ENS App 界面，诱导连接钱包 | **只使用** `app.ens.domains` |
| **免费空投** | 声称给你免费 .eth 域名，要求签名 | 不签署不了解的交易 |
| **过期通知欺诈** | 发送邮件称你的域名即将过期，引导到假网站 | 直接去官网检查，不要点击链接 |
| **虚假拍卖** | 3–4 位域名拍卖欺诈 | 通过官方平台参与拍卖 |
| **恶意签名** | 让你签署「域名转出」授权 | 每一笔签名前仔细核对内容 |

> ⚠️ **最重要的安全原则**：永远只在 `app.ens.domains` 上管理你的 ENS。如果你不确定一个网站是不是真的，**不要连接钱包**。详细安全知识请参考本站 [链上安全：签名授权与智能合约交互防骗指南](/onchain-security-signature-guide/)。

### 7.3 域名有效期查询

要查询你的 ENS 域名有效期：

1. 打开 [app.ens.domains](https://app.ens.domains)
2. 输入你的域名
3. 在域名详情页查看 **Expiry Date**（到期日期）
4. 或者直接使用 Etherscan 查询 NFT 的到期属性

### 7.4 域名隐私

- `.eth` 域名是链上公开数据——任何人都可以查到谁拥有 `yourname.eth`
- 如果你注重隐私，不要使用真实姓名注册 ENS
- 也可以使用多层级账户分离：一个「身份地址」持有 ENS 域名，另一个「操作地址」日常使用
