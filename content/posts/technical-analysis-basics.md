## 一、技术分析的核心思想

### 一句话理解

**技术分析（Technical Analysis）是通过研究历史价格和成交量数据，来预测未来价格走势的方法。** 它与基本面分析（研究项目团队、技术、生态等）是两条完全不同的路线。

技术分析建立在三个基本假设之上：

| 假设 | 含义 |
|:----|:----|
| **市场行为包容一切** | 所有消息面、基本面、情绪面的信息都已经反映在价格中 |
| **价格沿趋势运动** | 价格不会随机游走，而是会延续现有趋势一段时间 |
| **历史会重演** | 市场参与者的心理和行为模式会重复出现 |

### 技术分析 ≠ 玄学

> ⚠️ **重要提醒**：技术分析不是占卜，没有哪个指标能 100% 预测未来。技术分析的真正价值在于：
> - **提供概率优势**——让你在多次交易中胜率超过 50%
> - **制定交易计划**——知道什么时候进场、什么时候止损
> - **管理情绪**——用规则代替直觉，减少冲动交易

## 二、K 线图：技术分析的基础语言

### 2.1 什么是 K 线？

K 线（Candlestick，又称蜡烛图）起源于 18 世纪日本米市交易，是目前最主流的价格图表形式。**一根 K 线包含了四个关键价格信息：**

```
         最高价 (High)
    ┌─────────┐
    │   上影线   │
    ├──┬──┬──┤
    │  │  │  │  ← 收盘价（阳线）/ 开盘价（阴线）
    │  │  │  │
    │  │  │  │  ← 开盘价（阳线）/ 收盘价（阴线）
    ├──┴──┴──┤
    │   下影线   │
    └─────────┘
         最低价 (Low)
```

| 要素 | 含义 |
|:----|:----|
| **开盘价** | 该时间周期的起始价格 |
| **收盘价** | 该时间周期的结束价格 |
| **最高价** | 该时间周期内达到的最高价格 |
| **最低价** | 该时间周期内达到的最低价格 |
| **实体（Body）** | 开盘价与收盘价之间的部分 |
| **影线（Wick/Shadow）** | 最高/最低价与实体之间的部分 |

### 2.2 阳线与阴线

| 类型 | 颜色习惯 | 含义 |
|:----|:-------:|:----|
| **阳线（看涨）** | 🟢 绿色 / ⬜ 白色 | 收盘价 > 开盘价，买方占优势 |
| **阴线（看跌）** | 🔴 红色 / ⬛ 黑色 | 收盘价 < 开盘价，卖方占优势 |

### 2.3 常见 K 线形态

#### 单根 K 线形态

| 形态 | 图示 | 含义 |
|:----|:---:|:----|
| **大阳线** | 长实体，短影线或无影线 | 买方强势，上涨动力强 |
| **大阴线** | 长实体，短影线或无影线 | 卖方强势，下跌动力强 |
| **十字星** | 实体极短，上下影线长 | 多空势均力敌，可能出现反转 |
| **锤子线** | 下影线很长，实体小（底部） | 可能见底反弹 |
| **上吊线** | 下影线很长，实体小（顶部） | 可能见顶回落 |
| **倒锤子** | 上影线很长，实体小（底部） | 买方尝试上攻，可能反转向上 |
| **射击之星** | 上影线很长，实体小（顶部） | 冲高回落，可能见顶 |

#### 多根 K 线组合

| 组合形态 | 特征 | 含义 |
|:--------|:----|:----|
| **吞没形态** | 后一根 K 线实体完全覆盖前一根 | 强烈的反转信号 |
| **母子形态** | 后一根 K 线实体被前一根完全包含 | 趋势减弱，可能反转 |
| **三只乌鸦** | 连续三根大阴线 | 强烈看跌信号 |
| **红三兵** | 连续三根大阳线 | 强烈看涨信号 |
| **早晨之星** | 大阴线 + 十字星 + 大阳线 | 底部反转上涨 |
| **黄昏之星** | 大阳线 + 十字星 + 大阴线 | 顶部反转下跌 |

> 💡 **新手提示**：不要过度依赖单一 K 线形态，结合后续 K 线确认信号更可靠。一个形态的效力取决于它出现的位置（高位、低位还是震荡区间中）。

### 2.4 时间周期选择

不同的时间周期适合不同的交易风格：

| 周期 | 适合交易类型 | 单根 K 线代表时间 |
|:----|:----------:|:--------------:|
| **1 分钟 / 5 分钟** | 超短线 / 高频交易 | 1-5 分钟 |
| **15 分钟 / 1 小时** | 短线交易 / 日内交易 | 15 分钟 - 1 小时 |
| **4 小时** | 中短线交易 | 4 小时 |
| **日线** | 波段交易 / 趋势交易 | 1 天 |
| **周线 / 月线** | 长线投资 / 判断大趋势 | 1 周 - 1 月 |

> 💡 **推荐顺序**：新手从 **日线（1D）** 开始看，逐渐过渡到 **4 小时** 和 **1 小时**。不要直接看 1 分钟图——噪音太多，很容易被清洗出局。

## 三、趋势分析：判断当前处于什么市场

### 3.1 三大趋势类型

| 趋势 | 特征 | 对应的策略 |
|:----|:----|:---------|
| **上升趋势**（牛市） | 更高的高点和更高的低点 | 逢低买入，持有多单 |
| **下降趋势**（熊市） | 更低的高点和更低的低点 | 逢高卖出，持有空单 |
| **横盘震荡**（盘整） | 价格在区间内来回波动 | 高抛低吸，或观望等待突破 |

判断趋势的最简单方法：**连接两个以上的低点得到上升趋势线，连接两个以上的高点得到下降趋势线。**

### 3.2 支撑与阻力

**支撑位（Support）**：价格下跌到某个位置后开始反弹，这个位置就是支撑。
**阻力位（Resistance）**：价格上涨到某个位置后开始回落，这个位置就是阻力。

**如何找支撑和阻力？**

1. **前期高点和低点**：过去价格多次反转的位置
2. **整数关口**：如 BTC 的 $60,000、$70,000——心理价位
3. **移动平均线**：下文会详细介绍
4. **趋势线**：沿趋势方向画的斜线
5. **密集成交区**：过去成交量大的价格区间

**支撑和阻力的相互转化**：一个重要的支撑被跌破后，往往会变成新的阻力；同样，一个重要的阻力被突破后，往往会变成新的支撑。

## 四、最常用的技术指标

技术指标数量上百种，但新手不需要全部学。以下 5 个指标覆盖了 80% 以上的分析场景。

### 4.1 移动平均线（MA）

**移动平均线（Moving Average）是最基础、最实用的趋势指标。** 它计算一段时间内的平均价格，平滑掉短期的价格波动，让你看清趋势方向。

#### 常见均线周期

| 均线 | 周期 | 用途 |
|:----|:---:|:----|
| **MA5** | 5 日 | 超短线趋势，极为敏感 |
| **MA10/MA20** | 10-20 日 | 短线趋势参考 |
| **MA50/MA60** | 50-60 日 | 中线趋势，机构常用的「生命线」 |
| **MA120** | 120 日（约半年） | 中长期趋势 |
| **MA200/MA250** | 200-250 日（约年线） | 长期牛熊分界线 |

#### 两种主要均线

| 类型 | 计算方式 | 特点 |
|:----|:-------|:----|
| **SMA（简单移动平均线）** | 所有价格取等权平均 | 更平滑，反应稍慢 |
| **EMA（指数移动平均线）** | 近期价格权重更高 | 对价格变化反应更快 |

#### 均线的核心用法

**① 判断趋势方向**
- 短期均线 > 长期均线（多头排列）：上升趋势 📈
- 短期均线 < 长期均线（空头排列）：下降趋势 📉
- 均线纠缠在一起：震荡行情，无明显趋势

**② 金叉与死叉**

| 信号 | 含义 | 图示 |
|:----|:----|:----|
| **金叉** | 短期均线上穿长期均线（如 MA20 上穿 MA60） | 🟢 看涨信号 |
| **死叉** | 短期均线下穿长期均线（如 MA20 下穿 MA60） | 🔴 看跌信号 |

> ⚠️ **注意**：金叉和死叉是**滞后指标**——信号出现时价格往往已经走了一段。不要单靠金叉/死叉做交易决策，要结合价格位置和成交量判断。

**③ 均线作为支撑和阻力**
- 在上升趋势中，价格回踩 MA20/MA60 是常见的买入机会
- 在下降趋势中，价格反弹到 MA20/MA60 是常见的卖出机会
- MA200 被普遍认为是牛熊分界线：**价格站稳 MA200 之上 = 牛市，跌破 = 熊市**

### 4.2 RSI（相对强弱指标）

**RSI（Relative Strength Index）衡量价格变动的速度和幅度，帮助判断市场是否「过热」或「过冷」。**

| 参数 | 说明 |
|:----|:----|
| **默认周期** | 14（看 14 根 K 线的价格变化） |
| **取值范围** | 0-100 |
| **超买区** | RSI > 70，价格可能过高，有回调风险 |
| **超卖区** | RSI < 30，价格可能过低，有反弹机会 |
| **中间区** | RSI 30-70，正常波动区间 |

#### RSI 的实战用法

**用法 1：超买超卖**
- RSI > 70 → 市场过热，考虑减仓或做空（在震荡行情中有效）
- RSI < 30 → 市场过冷，考虑加仓或做多（在震荡行情中有效）

> ⚠️ **重要**：在强趋势行情中，RSI 可以长时间处于超买或超卖区域——**牛市中的 RSI > 70 可能只是上涨中继，熊市中的 RSI < 30 也可能继续下跌**。

**用法 2：背离（Divergence）——最强信号**

| 背离类型 | 现象 | 含义 |
|:--------|:----|:----|
| **顶背离** | 价格创新高，但 RSI 没有创新高 | ⚠️ 上涨动力减弱，可能见顶 |
| **底背离** | 价格创新低，但 RSI 没有创新低 | ⚠️ 下跌动力减弱，可能见底 |

> 💡 **背离是 RSI 最有价值的信号之一**，但需要等待确认——出现背离后，等价格实际反转再进场。

**用法 3：RSI 趋势线**
- 在 RSI 图上画趋势线，突破同样有参考价值
- 如果价格还在涨但 RSI 跌破趋势线，是提前的警告信号

### 4.3 MACD（指数平滑异同移动平均线）

**MACD 是结合趋势和动量的综合指标**，由三部分组成：

| 组成部分 | 说明 |
|:--------|:----|
| **DIF 快线** | 12 日 EMA - 26 日 EMA（代表短期趋势） |
| **DEA 慢线** | DIF 的 9 日 EMA（代表长期趋势） |
| **MACD 柱** | (DIF - DEA) × 2，红绿柱表示动量强弱 |

#### MACD 的三种核心用法

**① 金叉 / 死叉（与均线类似）**

| 信号 | 含义 |
|:----|:----|
| **DIF 上穿 DEA（零轴下方）** | 🟢 下跌趋势减弱，可能开始上涨 |
| **DIF 上穿 DEA（零轴上方）** | 🟢 上涨趋势加速，强势信号 |
| **DIF 下穿 DEA（零轴上方）** | 🔴 上涨趋势减弱，可能开始回调 |
| **DIF 下穿 DEA（零轴下方）** | 🔴 下跌趋势加速，弱势信号 |

**② 零轴（多空分界线）**
- MACD 在零轴上方 → 多头市场
- MACD 在零轴下方 → 空头市场
- DIF 从下方上穿零轴 → 趋势由空转多
- DIF 从上方下穿零轴 → 趋势由多转空

**③ MACD 背离**
- **顶背离**：价格创新高，但 MACD 红柱高度或 DIF 高点降低 → 🔴 看跌
- **底背离**：价格创新低，但 MACD 绿柱深度或 DIF 低点抬高 → 🟢 看涨

> 💡 **MACD 和 RSI 的背离信号互相验证时，信号的可靠性大幅提高。**

### 4.4 布林带（Bollinger Bands）

**布林带由三条线组成，展示价格的波动范围。**

| 组成部分 | 含义 |
|:--------|:----|
| **中轨** | 20 日简单移动平均线（SMA） |
| **上轨** | 中轨 + 2 倍标准差（价格过高） |
| **下轨** | 中轨 - 2 倍标准差（价格过低） |

#### 布林带的实战用法

**① 带宽收窄 → 变盘信号**
- 布林带上下轨大幅收窄（称为「缩口」）→ 市场即将迎来大幅波动
- 缩口越久，后续的突破幅度通常越大
- ⏳ **耐心等待缩口后的方向选择**，不要提前猜测方向

**② 价格触及上下轨**
- 价格触及上轨 → 价格偏高（但在强趋势中可能继续在上轨外运行）
- 价格触及下轨 → 价格偏低（但在强下跌趋势中可能继续在下轨外运行）

**③ 中轨作为趋势参考**
- 价格在布林带中轨上方运行 → 上升趋势
- 价格在布林带中轨下方运行 → 下降趋势
- 价格反复穿越中轨 → 震荡行情

> 💡 **布林带最有价值的用法**：经过长时间缩口后，价格放量突破上轨或跌破下轨，往往意味着新一轮趋势的开始。

### 4.5 成交量（Volume）

**成交量是技术分析中最被低估的指标。** 它告诉你价格变动背后有多少「真金白银」在支撑。

#### 成交量的核心原则

| 原则 | 含义 |
|:----|:----|
| **量价齐升** | 价格上涨伴随成交量放大 → 上涨健康，可持续 |
| **缩量上涨** | 价格上涨但成交量萎缩 → 上涨动力不足，可能反转 |
| **放量下跌** | 价格下跌伴随成交量放大 → 恐慌性抛售，可能继续下跌 |
| **缩量下跌** | 价格下跌但成交量萎缩 → 抛压减弱，可能接近底部 |
| **放量突破** | 价格突破关键阻力位且成交量放大 → 突破有效 |
| **缩量突破** | 价格突破关键阻力位但成交量小 → 可能是假突破 |

#### OBV（能量潮）

OBV 是成交量的累计指标：价格上涨日 +当日成交量，价格下跌日 -当日成交量。OBV 可以帮助判断成交量是否支持当前价格趋势。

- OBV 创新高而价格未创新高 → 资金提前入场，价格可能跟涨
- OBV 创新低而价格未创新低 → 资金提前离场，价格可能跟跌

## 五、实战：如何结合指标做分析

### 5.1 完整分析流程

不要依赖单一指标。推荐的「多指标交叉验证」流程：

```
第一步：看大趋势（日线）
  ├── MA200 → 判断牛熊
  └── 画趋势线 → 确认上升/下降/震荡

第二步：看中短期趋势（4H/1H）
  ├── MA20 + MA60 → 确认短期方向
  └── 布林带 → 判断波动状态（缩口还是开口）

第三步：看动量（同一周期）
  ├── RSI → 是否超买/超卖，有无背离
  └── MACD → 金叉/死叉，零轴位置，有无背离

第四步：看成交量验证
  └── 当前价格变动是否有成交量配合

第五步：制定交易计划
  ├── 入场点（价格到达什么位置）
  ├── 止损点（跌破什么位置离场）
  └── 止盈点（达到什么目标获利了结）
```

### 5.2 实战案例分析

**场景：判断 BTC 日线是否值得买入**

假设你看到以下情况：

| 指标 | 当前状态 |
|:----|:--------|
| **MA200** | 价格在 MA200 上方，趋势偏多 |
| **MA20 / MA60** | MA20 刚刚上穿 MA60（金叉形成） |
| **布林带** | 之前缩口后刚放量突破中轨 |
| **RSI (14)** | 55——中性区域，不超买也不超卖 |
| **MACD** | DIF 在零轴上方且上穿 DEA（看涨金叉） |
| **成交量** | 突破中轨时显著放量 |

**综合判断**：多个指标指向看涨方向，金叉 + 布林带放量突破 + RSI 中性仍有空间 → **可以制定入场计划，在回踩 MA20 时买入，止损设在 MA60 下方。**

### 5.3 新手常见误区

| 误区 | 正确做法 |
|:----|:--------|
| **只看一个指标做决策** | 至少 3 个指标互相验证 |
| **在 1 分钟图上频繁交易** | 新手从日线 / 4H 开始 |
| **指标越多越好** | 3-4 个经过验证的指标足够 |
| **看到金叉就全仓买入** | 等待回踩确认后再入场 |
| **忽视成交量验证** | 所有价格信号必须有成交量配合 |
| **大趋势向下时逆势做多** | 顺势而为，不要试图抄底 |
| **亏损后加仓摊平** | 按计划止损，不要加仓救单 |
| **过度优化历史参数** | 简单参数在实盘中往往更有效 |

## 六、总结与行动清单

### 一句核心总结

**技术分析的价值不在于预测未来，而在于为你提供一个客观的交易框架——在什么位置进场、在哪止损、何时离场。** 没有这个框架，交易就是赌博。

### 新手学习路径

| 阶段 | 学习内容 | 预计时间 |
|:----|:--------|:-------:|
| **第 1 周** | K 线图基础 + 趋势线与支撑阻力 | 掌握最核心的基础 |
| **第 2 周** | 移动平均线（MA）+ 多周期分析 | 学会判断趋势 |
| **第 3 周** | RSI + MACD + 背离识别 | 学会判断买卖点 |
| **第 4 周** | 布林带 + 成交量分析 | 结合判断突破真假 |
| **第 5 周** | 建立自己的分析流程 + 模拟交易 | 形成交易系统 |
