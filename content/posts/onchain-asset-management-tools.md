## 一、什么是链上资产管理工具？

### 一句话理解

**链上资产管理工具是一个「一站式看板」，把你的所有链上资产集中展示在一个界面上。** 你只需要连接一次钱包地址（或导入地址），它就会自动扫描该地址在**所有公链**和**所有 DeFi 协议**上的资产情况。

### 核心能力

| 功能 | 说明 |
|:----|:----|
| **多链资产总览** | 一个页面看到你在 Ethereum、Arbitrum、Base、Optimism、BSC、Polygon 等十几条链上的所有代币余额 |
| **DeFi 头寸追踪** | 自动识别你在 Uniswap、Aave、Compound 等协议中的 LP、借贷、质押头寸，并显示当前价值 |
| **交易历史** | 按时间线展示你所有的链上交互记录，包括 Swap、转账、授权等 |
| **NFT 展示** | 自动识别钱包中的 NFT 资产并显示地板价和稀有度信息 |
| **Gas 监控** | 实时显示当前各链 Gas 价格，便于选择最佳操作时机 |
| **钱包健康检查** | 分析钱包的风险状况（授权过多、与高风险合约交互等） |

### 为什么你需要它？

- **省时间**：不用逐个切换网络和打开不同 DApp 查看资产
- **防遗漏**：不会忘记你在某个协议里的质押或 LP 头寸
- **更方便**：随时了解你的链上总资产净值，用作记账或税务统计
- **更安全**：可以直接看到哪些合约有你的授权，及时发现异常

## 二、三大主流资产管理工具对比

目前链上资产管理工具市场由三个平台主导——DeBank、Zapper 和 Zerion。它们各有特色，侧重点不同。

| 对比项 | **DeBank** | **Zapper** | **Zerion** |
|:------|:---------:|:---------:|:---------:|
| 上线时间 | 2019 | 2018 | 2019 |
| 支持链数 | **100+** | **20+** | **20+** |
| 是否开源 | 否 | 部分开源 | 否 |
| 原生代币 | 无 | 无 | ZERO（有治理/质押功能） |
| 最佳用途 | **最全面的多链资产扫描** | **DeFi 交互入口** | **移动端体验最佳** |
| 是否需要注册 | 不需要（直接查地址） | 需要（邮箱/钱包登录） | 需要（邮箱/钱包登录） |
| 专业版 | 免费 | 免费 + Zapper Pro（$9.99/月） | 免费 + Zerion Premium |

### DeBank —— 覆盖面最广的「多链浏览器」

**核心优势**：支持的链最多（100+），扫描最全面，还有独特的钱包地址分析功能。

DeBank 最初是一个 DeFi 协议数据聚合平台，后来发展为最全面的多链资产追踪工具。它最大的特点是**不需要注册或连接钱包**——你只需要输入一个钱包地址，就能查看该地址的所有链上资产。

**适合人群**：需要在多条链间操作的高级用户、希望匿名查询地址的投资者。

### Zapper —— 最强 DeFi 交互入口

**核心优势**：不仅能看，还能直接在平台上进行 DeFi 操作（存入、取出、兑换）。

Zapper 前身是 DeFiSnap，最初专注于 DeFi 头寸展示。它最大的特色是**可操作**——你可以在 Zapper 上直接进行 Swap、添加/移除流动性、存入/提取借贷协议等操作，无需跳转到各个 DApp 页面。

**适合人群**：DeFi 重度用户、需要在同一界面管理多个协议头寸的用户。

### Zerion —— 移动端体验之王

**核心优势**：App 体验最好（iOS/Android），UI 设计精美，操作流畅。

Zerion 最初是 DeFi 投资组合追踪工具，后来推出了自己的钱包功能（Zerion Wallet），拥有完整的移动端体验。Zerion 的 Premium 订阅提供了更多高级功能，如实时推送通知、高级筛选等。

**适合人群**：习惯在手机上管理资产的用户、注重 UI 体验的用户。

## 三、实操：DeBank 完整使用教程

DeBank 覆盖面最广且无需注册，最适合作为入门选择。下面以 DeBank 为例进行详细操作演示。

### 3.1 打开 DeBank

访问 [debank.com](https://debank.com)，你会看到一个简洁的搜索框。

> ✅ DeBank **不需要注册账号**，也不需要连接钱包。只需要输入区块链地址即可查询。

### 3.2 搜索钱包地址

在搜索框中输入你的钱包地址（以 `0x` 开头的以太坊系地址），回车。

**示例**：
- 你可以输入你的 MetaMask 地址
- 也可以输入任意公开地址来查看其资产——所有链上数据都是公开的

### 3.3 主界面解读

搜索进入后，你会看到几个核心板块：

#### 资产总览（Portfolio）

页面顶部展示你所有链上的**总资产净值**，按 USD 计价。下方是按链分类的代币列表：

```
总资产净值：$12,345.67

Ethereum        $5,200.00    ETH: 1.5 | USDC: 500 | UNI: 100
Arbitrum        $3,100.00    ETH: 0.8 | USDC: 1,000
Base            $2,500.00    ETH: 0.5 | AERO: 1,000
Optimism        $1,200.00    ETH: 0.3 | OP: 200
BNB Chain       $345.67      BNB: 1
```

> 💡 **提示**：DeBank 会自动将各类代币（包括 LP 代币、质押代币）的价值折算为 USD，直接在总资产中体现。

#### DeFi 头寸（Protocol）

这是 DeBank 最强大的功能之一。它会自动识别你钱包中**所有 DeFi 协议**中的头寸，包括：

- **流动性池（LP）**：你在 Uniswap、Curve、Aerodrome 等 DEX 提供的流动性
- **借贷头寸**：你在 Aave、Compound 等平台上存入的资产和借出的资产
- **质押（Staking）**：你在各种协议中质押的代币
- **收益聚合器**：你在 Yearn、Beefy 等收益平台中的存款

每个头寸都会显示：
- 存入资产数量及当前 USD 价值
- 当前收益率（APR/APY）
- 已获得的奖励
- 近 7 天/30 天的收益变化

**示例 — 查看 Uniswap LP 头寸**：
```
Uniswap V3 (Arbitrum)
├── ETH/USDC 0.05%
│   ├── ETH: 2.5 ($8,250)
│   ├── USDC: $7,800
│   ├── 未收手续费: $12.50
│   └── 收益率: 15.2% APR
└── ARB/ETH 0.30%
    └── ... 
```

#### 交易历史（History）

按时间倒序展示你所有链上的交易记录：

```
2026-07-11 20:15  → Swap 0.5 ETH → 1,500 USDC     Arbitrum    ✅
2026-07-11 14:30  → Approve USDC for Uniswap       Arbitrum    ✅
2026-07-10 22:00  → Provide ETH/USDC LP            Base        ✅
```

每条记录都可以点击，直接跳转到对应的区块浏览器查看详情。

#### 代币授权（Token Approvals）

这是一个非常重要的安全功能。**「代币授权」** 页面会列出你曾经 Approve 过的所有智能合约——即你授权过的、可以用你钱包里的某种代币扣款的合约。

```
授权代币      被授权合约            额度              状态
───────     ─────────           ─────             ────
USDC        0xUniswapRouter  →  无限额            正常
USDC        0xOldProtocol    →  10,000 USDC       ⚠️ 已废弃，建议撤销
USDC        0xUnknownDApp    →  无限额             🚨 高风险！立即撤销
```

> ⚠️ **安全提示**：定期检查 Token Approvals 页面，**撤销对不再使用的智能合约的授权**是保护资产安全的重要手段。如果有未知合约获得了「无限额」授权，应立即撤销。关于授权的安全风险详细解读，请参考本站 [链上安全：签名授权与智能合约交互防骗指南](/onchain-security-signature-guide/)。

### 3.4 实用进阶功能

**多地址组合（Portfolio Groups）**：
DeBank 支持将多个钱包地址组合成一个「投资组合」，查看所有地址的总资产。适合管理多个钱包的用户。

**价格提醒**：
在 DeBank 中关注特定代币后，可以设置价格变动提醒（通过 Telegram 或邮件）。

**链上数据看板**：
DeBank 的「热门协议」页面展示各协议的关键数据，包括 TVL、交易量、活跃用户数等，是研究新协议的重要参考。

## 四、Zapper 使用教程：从查看到操作

如果说 DeBank 是「只看不动」，那 Zapper 就是「边看边做」。

### 4.1 Zapper 的特点

Zapper 的界面和 DeBank 类似——输入地址后展示多链资产总览。但 Zapper 允许你**直接在平台上操作**：

- **Swap**：直接兑换代币（聚合了多个 DEX 的流动性）
- **Deposit/Withdraw**：直接在界面上存入或提取借贷协议头寸
- **Add/Remove Liquidity**：管理你的 LP 仓位
- **Bridge**：在支持的链之间跨链转账

### 4.2 核心操作演示

#### 在 Zapper 上执行 Swap

1. 打开 [zapper.fi](https://zapper.fi)，连接你的钱包（Zapper 需要连接钱包才能操作）
2. 点击顶部的 **Swap** 按钮
3. 选择源链和目标链（例如在 Arbitrum 上换币）
4. 输入你持有的代币和想兑换的代币（例如 ETH → USDC）
5. Zapper 会自动聚合多个 DEX 的报价，给出最佳路径和最低滑点
6. 确认交易，钱包签名即可完成

> ✅ **优势**：Zapper 会自动比较 Uniswap、Curve、Balancer 等多个 DEX 的价格，给出最优的兑换路径，通常比单独使用某个 DEX 更划算。

#### 管理 LP 头寸

在 Zapper 的 Portfolio 页面，点击你的 LP 头寸，可以直接：
- **添加流动性**：增加 LP 份额
- **移除流动性**：提取 LP 中的代币
- **领取手续费**：收累计的交易手续费分成
- **Stake LP 代币**：将 LP 代币质押到对应的收益池中

### 4.3 Zapper Pro

Zapper Pro 是付费版本（$9.99/月），提供：
- 实时价格推送（WebSocket，无需手动刷新）
- 自定义标签和备注
- 更详细的收益分析和税务报告
- 多个钱包地址的汇总对比

> 💡 **是否需要付费？** 普通用户免费版完全够用。如果你管理较大规模资产（$50K+）或经常做 DeFi 操作，Pro 的实时推送功能比较实用。

## 五、Zerion 使用教程：移动端首选

### 5.1 Zerion 的特色

Zerion 是三个工具中**移动端体验最佳**的——它的 iOS 和 Android 应用非常流畅，UI 设计精美，操作逻辑清晰。

**它和 DeBank/Zapper 最大的区别**：
1. **自带钱包功能**：Zerion 有自己的内置钱包（Zerion Wallet），可以在 App 内直接创建或导入钱包
2. **原生移动端体验**：针对手机操作做了大量优化，手势操作流畅
3. **推送通知**：当钱包发生大额交易时，会实时推送通知到手机

### 5.2 安装与使用

1. 在 App Store 或 Google Play 搜索 **Zerion** 下载
2. 可以选择「连接已有钱包」（WalletConnect 或导入私钥）或「创建新钱包」
3. 连接后自动扫描所有链上的资产

### 5.3 独家功能

**实时推送通知**：
当你的钱包发生以下事件时，Zerion 会推送通知：
- 大额转账（入账或出账）
- 授权（Approve）操作
- 头寸清算预警
- 代币价格大幅波动

> 💡 这个功能在**安全防护**上非常有用——如果有人获得了你的 Approve 权限准备转走资产，你会在第一时间收到通知，从而有时间将剩余资产转移到安全地址。

**WalletConnect 一键连接**：
Zerion App 内置了 WalletConnect 功能，可以直接在手机上连接各种 DApp 进行操作。

## 六、三款工具如何选择？

### 按使用场景推荐

| 你的需求 | 推荐工具 | 理由 |
|:--------|:-------:|:----|
| 只想快速查看一个地址的资产 | **DeBank** | 无需注册，输入地址即可查询 |
| 在多条链之间频繁操作 | **Zapper** | 内置 Swap、LP 管理、跨链功能 |
| 主要用手机管理资产 | **Zerion** | App 体验最佳，推送通知实用 |
| 需要分析风险/撤销授权 | **DeBank** | Token Approvals 功能最全面 |
| 深度 DeFi 用户 | **Zapper + DeBank** | Zapper 操作，DeBank 查授权 |
| NFT 收藏者 | **Zerion / DeBank** | 两者都有较好的 NFT 展示界面 |

### 比较建议

**我的推荐是：三个都用。** 它们都是免费的（基础功能），各自的侧重点不同：

1. **用 DeBank 做日常资产总览**——覆盖面最广，打开网站输入地址就能看
2. **用 Zapper 做 DeFi 操作**——需要执行 Swap 或管理 LP 时用 Zapper
3. **用 Zerion 做移动端监控**——手机上安装，随时收到资产变动通知

三者之间没有互斥关系，它们读取的是同一组链上数据，只是在展示和交互方式上各有侧重。

## 八、安全须知

### 8.1 使用资产管理工具是否安全？

**简单回答：非常安全，可以放心使用。**

- DeBank、Zapper、Zerion 都是**只读或只签名**的工具
- 它们**不会持有你的私钥**，也不会主动转走你的资产
- Zapper/Zerion 虽然支持「操作」功能，但每笔操作都需要你在钱包中签名确认——和直接在 DApp 上操作一样安全

> ⚠️ **唯一需要警惕的是钓鱼网站。** 确保你访问的网址是正确的：
> - DeBank：`debank.com`（不是 `debank.pro` 或其他变体）
> - Zapper：`zapper.fi`（不是 `zapper.app` 等）
> - Zerion：`zerion.io`（注意拼写）

### 8.2 常见误区

| 误区 | 真相 |
|:----|:----|
| 「这个平台能查到我的所有资产，是不是能转走我的钱？」 | 不能。查询是只读操作，你的私钥在钱包里，未经签名的任何操作都无法执行。 |
| 「连接资产管理工具是不是在授权？」 | 连接钱包 ≠ 授权（Approve）。连接只是让平台读取你的地址。但注意：在 Zapper 上执行 Swap 或存款时，会有标准的智能合约交互——和你在 Uniswap 上操作一样。 |
| 「这个平台要我的私钥/助记词？」 | **任何资产管理工具都不应该要求你的私钥或助记词。** 如果有人索要，100% 是钓鱼。 |


### 8.3 合规与隐私提示

链上资产管理工具读取的是**公开的链上数据**。任何人在 Etherscan 上输入你的地址也能看到你的资产——资产管理工具只是把数据整理成更容易看懂的格式。

**但要注意以下隐私风险：**
- 如果你在资产管理工具上连接了钱包（Zapper/Zerion），平台会记录你的 IP 地址和钱包地址的关联
- 如果你只是想匿名查询，使用 DeBank 的搜索功能（不连接钱包）即可
- 对于大额持仓用户，建议不要在任何工具上登录或连接钱包，单纯用地址搜索查看
