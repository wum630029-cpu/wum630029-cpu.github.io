## 一、现货交易

现货交易是最基础、最适合新手的交易方式。你在 OKX 买入的加密资产会直接存入你的现货钱包。

### 如何开始现货交易？

1. 打开 OKX App → 底部点击「交易」
2. 搜索并选择交易对（如 BTC/USDT）
3. 选择订单类型 → 输入数量 → 确认交易

### OKX 现货订单类型

| 订单类型 | 说明 | 适用场景 |
|:-------|------|:--------|
| 限价单 | 设定指定价格挂单，达到价格后成交 | 计划交易、等回调买入 |
| 市价单 | 以当前最优价格立即成交 | 快速进出场 |
| 高级限价 | 可选「只做 Maker」或「全部成交或撤销」 | 专业交易 |

### 交易界面说明

OKX 的交易界面分为几个区域：

- **顶部**：当前价格和 24 小时涨跌幅
- **K 线图**：展示价格走势，支持 1 分钟到月线多种周期
- **盘口**：左侧买盘（绿色）、右侧卖盘（红色）
- **下单区**：选择类型、输入价格和数量
- **持仓区**：查看当前持有的资产和订单

> **新手建议**：先从 BTC/USDT 和 ETH/USDT 这两个主流交易对开始，深度好、波动相对可控。

## 二、永续合约交易

OKX 的永续合约是其旗舰产品之一，深度好、支持主流币种多。

### 开通合约账户

1. App 底部「交易」→ 切换到「合约」
2. 阅读合约协议 → 同意开通
3. 从现货钱包转入 USDT 作为保证金

### OKX 合约的特色功能

**1. 统一账户**

OKX 的**统一账户**是其核心特色——你不需要把资金分开到现货和合约账户，一个账户统一管理所有资金。这带来几个好处：

- 资金利用率更高
- 一个账户就能做套利（对冲）
- 查看资产总览更方便

**2. 全仓与逐仓**

| 模式 | 风险 | 适合 |
|:---|:----|:----|
| 逐仓 | 每笔仓位独立，爆仓不牵连其他 | 新手、每笔单独控制风险 |
| 全仓 | 账户余额作为所有仓位的保证金 | 进阶、套利和对冲 |

**3. 杠杆选择**

OKX 永续合约支持 **1x-125x** 杠杆选择。新手建议 **2x-5x**，不要追求高倍数。

### 合约开仓流程

1. 选择交易对（推荐 BTC/USDT 或 ETH/USDT 永续）
2. 设置杠杆倍数（新手建议 **3x**）
3. 选择保证金模式（**逐仓**更安全）
4. 输入开仓数量和方向
5. 开仓后立刻设置**止损单**

**每单必带止损！** 不带止损的合约等于赌博。

### 资金费率

OKX 永续合约的资金费率和币安类似，每 8 小时结算一次。当多头过多时多头付空头，空头过多时空头付多头。正常范围在 ±0.01%。

## 三、跟单交易

跟单交易是 OKX 的特色功能之一，**非常适合新手和没时间盯盘的交易者**。

### 什么是跟单交易？

跟单交易允许你**一键复制专业交易员的策略**。你只需要选择你信任的交易员，设定跟单金额，系统就会自动同步该交易员的开仓和平仓操作。

> 简单说：你选一个厉害的人，他买你也买，他卖你也卖，完全自动执行。

### 如何选择合适的交易员？

OKX 跟单页面会展示交易员的详细数据：

| 指标 | 说明 | 参考值 |
|:----|------|:------|
| 总收益率 | 该交易员的总盈利比例 | 100%+ → 优秀 |
| 交易天数 | 活跃交易的天数 | >90 天 → 有经验 |
| 胜率 | 盈利交易占比 | >60% → 稳健 |
| 最大回撤 | 历史最大亏损比例 | <20% → 风控好 |
| 跟单人数 | 当前跟随者数量 | 多 → 受认可 |

**选择标准建议：**

1. **交易时间 >3 个月** — 经过市场检验
2. **最大回撤 <20%** — 风控做得好
3. **收益率 >50%** — 盈利能力有数据支撑
4. **不要只看收益率** — 回报越高往往风险越大

### 设置跟单参数

选定交易员后，你可以设置：

- **固定投资金额**：每次自动复制交易的金额
- **最大持仓数量**：最多同时持有几个跟单仓位
- **跟单比例**：按交易员仓位的百分比跟随（如 10%）

### 跟单注意事项

| 注意点 | 说明 |
|:------|------|
| 不代表稳赚 | 交易员也有亏损期，过去业绩不代表未来 |
| 需要关注 | 定期查看跟单情况，表现不好及时停止 |
| 自行设止损 | 部分跟单支持设置全局止损 |
| 分散跟单 | 建议跟 2-3 个不同风格的交易员，不要全押一个 |

## 四、OKX 进阶功能

### 1. 策略交易

OKX 内置多种交易策略：

- **网格交易**：在震荡行情低买高卖，自动执行
- **DCA 定投**：定期定额买入，降低平均成本
- **冰山委托**：大额订单分成小单成交，减少市场冲击

### 2. 期权交易

OKX 提供 BTC 和 ETH 的期权交易，支持做多/做空，适合专业交易者。新手建议先了解清楚再尝试。

### 3. 永续合约网格

将网格策略应用到合约交易，可以做多、做空或双向。适合对网格交易有经验的用户。

## 六、常见问题 FAQ

### Q1：OKX 最小交易量是多少？

现货最小取决于交易对，BTC/USDT 最小可买 0.00001 BTC（约 0.6 USDT）。合约最小开仓约 5 USDT。

### Q2：OKX 支持一键平仓吗？

支持。持仓页面点击「一键平仓」即可按市价全部平仓。建议在行情剧烈波动时使用。

### Q3：跟单交易会亏损本金吗？

会。跟单是复制别人的交易，亏损时你的资金也会亏损。跟单不保本，不保障收益。

### Q4：OKX 合约交易有模拟盘吗？

有。OKX 提供**模拟交易功能**，可以使用虚拟资金练习合约交易，**强烈建议新手先模拟再实盘**。

### Q5：OKX 支持 API 量化交易吗？

支持。OKX 提供完善的 REST API 和 WebSocket API，支持现货、合约、期权等。文档齐全，适合有编程能力的用户。
